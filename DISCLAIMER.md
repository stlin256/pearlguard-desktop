# Disclaimer

PearlGuard Desktop is an independent desktop companion project. It is not an official Pearl Wallet product and is not financial advice.

Cryptocurrency transactions are irreversible. Mining pool APIs can be incomplete, unavailable, delayed, or inconsistent. Verify wallet addresses, thresholds, fees, pool data, and local settings before using any automation with real funds.

The preview release uses dry-run behavior for transfer checks and does not broadcast transactions during automated tests.
